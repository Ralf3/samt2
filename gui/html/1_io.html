3.1 Input / Output

HDF Open reads all maps from an HDF5-file
HDF Save Grid saves a highlighted grid with: P1=gridname, P2=model, P3=modeler
ASCII Open opens a highlighted ASCII grid
ASCII Save saves a highlighted ASCII grid
POINT Open geo reads in a point theme with x,y,z in geographic coordinates
POINT Open ij reads in a point theme with i,j,z in matrix coordinates
FUZZY Open reads in a fuzzy model
SVM Open reads in a trained SVM model
Exit ends SAMT2
