Open a fuzzy model - Layout of an input file

SAMT2 - Fuzzy is able to open files with ending *.fis or *.fis1. But this files must have a special structure so the tool will be able to create the fuzzy model in a correct way. If you do not adhere to this structure, SAMT2 - Fuzzy will not correctly open the fuzzy model. In the following part, the layout of a fis-file will be explained.

input input1 5
member small triangular
-1 10 20
member small_medium triangular
10 20 30
member medium triangular
20 30 40
member medium_hot triangular
30 40 50
member hot triangular
40 50 65
outputs temperature 5
output cold 10
output cold_medium 25
output medium 35
output medium_hot 45
output hot 60
rules 5
0 4 1.0
1 3 1.0
2 2 1.0
3 1 1.0
4 0 1.0

The file interpreter of SAMT2 - Fuzzy will be controlled by a fis file with the following structure:

  • Inputs
  • Outputs
  • Rules

The inputs must begin with the keyword input, the name of the input and the number of membership functions. There are up to three inputs possible. The membership function can be of type trapeze or triangular. For specifying the trapeze type there were expected 4 numbers (the lower left point, the upper left point, the upper right point and the lower right point), for specifying the triangular type we only need 3 numbers.

In that example there are specified five membership functions (from cold to hot) for the output temperature.

The rules combine the inputs with the outputs by numbering. In our example we have only one input. That means that the rule consists only of three columns: input, output, cf
