1 Overview of the functions in SAMT2

1.1 IO

HDF_Openreads all maps from an HDF5-file
HDF_Save_Gridsaves a highlighted grid with: P1=gridname, P2=model, P3=modeler
ASCII_Openopens a highlighted ASCII grid
ASCII_Savesaves a highlighted ASCII grid
POINT_Opengeoreads in a point theme with x,y,z in geographic coordinates
POINT_Openijreads in a point theme with i,j,z in matrix coordinates
Fuzzy_Openreads in a fuzzy model
SVM_Openreads in a trained SVM model
Exitends SAMT2

1.2 Tools

FuzzyOpens the fuzzy development tool
Fuzzy_GeneratorOpens the fuzzy generator to train a fuzzy model with data
SadatoOpens the sadato development tool

1.3 Analysis

Histogramshows a histogram of the highlighted grid with P1=bins=20
Statisticsshows statistics in a separate window: P1=min, P2=max, P3=std
Infoshows in: P1=nrows, P2=ncols, P3=nodata
MakeMapshows an Open Street-map map of the highlighted grid
Resizeresizes a grid using splines, P1=nrows, P2=ncols
Corrcorrelates the highlighted with the grid in P1: P2=corr
Samplesamples P1=100 random points from the highlighted grid
Sample_detsamples P1=val all points for mat[i,j]==val
Export_Rsamples 2000 data from P1=g1, P2=g2, P3=g3 and the highlighted as train

1.4 Simple Grid

Setset mat=P1, (overwrite)
Replacereplaces the value P1 with the value P2, (overwrite)
Addmat+=P1, (overwrite)
Mulmat*=P1, (Overwrite)
Logcalculates the lg(P1+x) for all x>=0, (overwrite)
Lncalculates the ln(P1+x) for all x>=0, (overwrite)
Fabscalculates the fabs(mat), (overwrite)
Normmat=(mat-gridmin)/(gridmax-gridmin), (overwrite)
ZNormmat=(mat-mean)/std, (overwrite)
Invcalculates: gridmax-mat+gridmin, (overwrite)
Inv_acalculates: a-mat, P1=a , (overwrite)
Condif(mat<P1) mat=P1, if(mat>P2) mat=P3, (overwrite)
Cutclamps the mat: if(mat>P1) mat=P2=nodata (overwrite)
Cutoffsets all values in the range [P1,P2) to P3=nodata, (overwrite)
Classclassifies the continuous grid into n P1=10 discrete classes
Rand_Intmat=randint(P1=0,P2=10), (overwrite)
Rand_Floatmat=randfloat(), (overwrite)

1.5 Advanced Grid

Combine_AddP1=g1: mat=mat+g1, (overwrite)
Combine_DiffP1=g1: mat=mat-g1, (overwrite)
Combine_MulP1=g1: mat=mat*g1, (overwrite)
Combine_MinP1=g1: mat=min(mat,g1) (overwrite)
Combine_MaxP1=g1: mat=max(mat,g1) (overwrite)
Combine_OrP1=g1: if(mat==nodata) mat=g1 (overwrite)
Combine_AndP1=g1: if(g1.mat==nodata) mat=nodata (overwrite)
Flood_Fillfloods a region: mouse=(i,j), P1=level, P2=mark (overwrite) return counts
Flood_Fillstdfloods a region: mouse=(i,j), P1=level, P2=mark (overwrite) return std
Varpartcreates a new grid with minimum variance clustering P1=5000 cluster
Remove_Trendremoves the trend to find local minmax: using P1=2000 sample points (overwrite)
Grad_d4calculates the gradient of a d4 environment
Grad_d8calculates the gradient of a d8 environment
Gen_Maskgenerates a mask from a grid using the mouse

1.6 Kernel

Kernel_sciGaussian kernel with P1=height, P2=width, P3=sigma=1, (overwrite)
Kernel_squrectangular kernel with P1=height, P2=width, (overwrite)
Kernel_circircular kernel with P1=radius, (overwrite)
Knnk nearest neighbor: P1=k, P2=min, P3=max, (overwrite)

1.7 Points

InterpolateP1=grid, highlighted=points
Poissonsolves the Poisson equation: highlight=points, P1=grid, P2=eps, P3=iter
Voronoicalculates a voroni map from a highlighted=points, P1=grid
Distancecalculates the euclidean distance between the highlighted=points, P1=grid

1.8 View

Colorbaropens a separate window with a colorbar P1=min, P2=max optional
Show_Diffshows the difference between the highlighted and the P1
Show_3dshows a 3d view with a stride P1=10 (larger strides: lower resolution)
Show_Rangeshow only values between P1=min and P2=max
Show_Listshow a list used to select only discrete values of a grid
Show_BlackWhiteswitches between bw and color

1.9 Table

Lutreplaces discrete values mat with new values (overwrite)
Selectallows to select discrete values using a tabular (overwrite)
