3.8 View

Colorbar opens a separate window with a colorbar, optional P1=min, P2=max
Show_Diff shows the difference between the highlighted and the P1
Show_3d shows a 3d view with a stride P1=10 (larger strides: lower resolution)
Show_Range show only values between P1=min and P2=max
Show_List show a list used to select only discrete values of a grid
Show_Black_White switches between bw and color
