Installation Guide

The GUI is written in Python using the QT libraries. Please install the following additionally Python packages:
  • subprocess
  • locale
  • PyQt4
  • pandas
  • mpl_toolkits
If you have downloaded SAMT2 from https://github.com/Ralf3/samt2, a compilation is necessary. Compile the source code of grid in samt2/src by typing 'make', then do the same with Pyfuzzy in samt2/fuzzy/src .

Start the SAMT2-Fuzzy tool by typing samt2fuzzy.py

Set the environment variables SAMT2MASTER and SAMT2DATEN

Open the file .bashrc with an editor and than add the following instruction:

  • export SAMT2MASTER=/path_to_samt2

  • export SAMT2DATEN=/path_to_data

  • PATH=$PATH:/$SAMT2MASTER/gui/:/$SAMT2MASTER/fuzzy/gui/:.:

Remark:
path_to_samt2  for the path where samt2 is installed
path_to_data  for the path where the data are installed
data contain :
ascii : dir for ASCII grid data from ARCGIS
hdf : dir for hdf files containing grids
data : dir for modeling data
points : dir for point data
model :
  svm : dir for SVM models
fuzzy : dir for fuzzy models
