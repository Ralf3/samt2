Set sets mat=P1, (overwrite)
Set_ND set all value exept nodata=P1, (overwrite)
Replace replaces the value P1 with the value P2, (overwrite)
Add mat+=P1, (overwrite)
Mul mat*=P1, (Overwrite)
Log calculates the lg(P1+x) for all x>=0, (overwrite)
Ln calculates the ln(P1+x) for all x>=0, (overwrite)
Fabs calculates the fabs(mat), (overwrite)
Norm mat=(mat-gridmin)/(gridmax-gridmin), (overwrite)
ZNorm mat=(mat-mean)/std, (overwrite)
Inv calculates: gridmax-mat+gridmin, (overwrite)
Inv_a calculates: a-mat, P1=a , (overwrite)
Cond if(matP2) mat=P3, (overwrite)
Cut clamps the mat: if(mat>P1) mat=P2=nodata (overwrite)
Cut_Off sets all values in the range [P1,P2) to P3=nodata, (overwrite)
